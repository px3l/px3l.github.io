Product Manager at BIMLauncher Ltd

BIMLauncher Ltd

BIMLauncher ltd develops and runs a cloud based set of integrations between software vendors APIs that allowed data to be moved from one vendor to another, essentially the ability to synchronise or migrate data between different cloud software. This is managed through a centralised hub, hosted online.

I came in at an early stage, where an initial integration had been completed and a new, more complex one had to be developed. I played an essential role in utilising various online methods of collaboration to successfully build and deliver this new integration for the client.

We had a team working remotely all over the world: our senior developer in Indonesia, the CEO in Ireland, I was in England and others in Canada, Philippines and Lebanon. Because of the time differences, we had to plan the few meetings we could have together via video chat and so we relied on using software which everyone could input to in their own working time.

I had to keep everyone on the same page, by ensuring our shared information was correct and easy for each other to understand by themselves. Alongside this, I had to help develop the integration from conceptual design to technical specifications which could be relayed to the team. The integration was delivered successfully and immediately went live on 16 projects.

Since then, my focus has been on a marketing strategy along with UX and site design.
